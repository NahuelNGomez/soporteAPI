# Requisitos
* Python 3.10
 
## Para correr la app:

### Instalar los requisitos:
`` pip install -r requirements.txt``

### Correr la app:
`` uvicorn app:app``

### Acceder a FastApi:
`` https://apisoporte.onrender.com/docs``

#### Correr test de cucumber:
`` behave --color test\features``

## Ejecución por script en windows 10:
`` ./run.ps1``
